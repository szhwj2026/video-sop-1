# 短视频自动化生产平台 - MVP 需求文档

**版本**：v1.0  
**日期**：2026-03-23  
**状态**：待评审

## 📋 项目概述

### 1.1 项目背景

为自媒体人打造一个**从选题到成片全流程自动化**的短视频生产平台，降低内容创作门槛，提高生产效率。

### 1.2 核心目标

**一句话**：输入一个话题或图片，自动产出 1 分钟短视频成片。

### 1.3 MVP 范围

| 阶段 | 目标 | 周期 |
|------|------|------|
| MVP | 图文类视频自动化生产 + 作品展示网站 | 1-2 周 |
| V2 | 增加真人/动画视频生成、多平台分发 | 4-6 周 |
| V3 | 数据看板、粉丝互动、商业化功能 | 8-12 周 |

## 🎯 核心功能模块

### 2.1 选题策划模块

**功能描述**：根据输入的话题/图片，自动生成选题分析

**输入**：
- 话题关键词（如"AI 助手"）
- 或参考图片

**输出**：
- 热点分析（当前相关热点事件）
- 行业洞察（行业趋势、痛点）
- 产品结合点（如何与产品关联）
- 推荐选题方向（3-5 个）

**技术实现**：
- Tavily API 抓取热点
- 阿里百炼 Qwen 分析输出

---

### 2.2 脚本生成模块

**功能描述**：根据选题，生成 1 分钟视频脚本

**输出格式**：
```
【开场】0-5s：钩子句，吸引注意力
【正文】5-45s：3 个核心观点，每点 10-15s
【金句】45-55s：总结性金句
【结尾】55-60s：互动引导 + 关注提示
```

**脚本要素**：
- 口语化表达（别/啥/咋等）
- 每段配画面描述
- 标注 BGM 节奏点
- 字幕文本同步输出

**技术实现**：
- 阿里百炼 Qwen（已有配置）
- 预设脚本模板

---

### 2.3 图片生成模块

**功能描述**：根据脚本，自动生成配图

**输出要求**：
- 封面图：1 张，16:9 横屏，800KB 以内
- 场景图：5-8 张，16:9 横屏，600KB 以内
- 图片风格：与账号定位一致

**技术实现**：
- 火山引擎 Doubao-Seedream-4.5（已有 API 配置）
- 脚本自动提取画面描述 → 生成提示词

---

### 2.4 视频剪辑模块

**功能描述**：将图片、配音、字幕、BGM 合成为 1 分钟视频

**输入**：
- 脚本（含时间轴）
- 配图（6-9 张）
- AI 配音音频
- BGM 音乐

**输出**：
- MP4 格式，1080x1920（竖屏）或 1920x1080（横屏）
- 时长：55-65 秒
- 大小：10-30MB
- 硬编码字幕

**技术实现**：
- ffmpeg 自动化脚本
- 预设转场效果（淡入淡出、缩放）
- 字幕 SRT 生成 + 烧录

---

### 2.5 AI 配音模块

**功能描述**：将脚本转为语音

**要求**：
- 自然度高的 AI 语音
- 支持语速调节
- 支持情感表达

**技术实现**（可选）：
- ElevenLabs（高质量，付费）
- 火山引擎 TTS（已有配置）
- Azure TTS

---

### 2.6 作品展示网站

**功能描述**：展示生成的短视频作品，支持播放、分类、分享

**页面结构**：
```
首页 → 作品列表（封面 + 标题 + 播放量）
作品详情页 → 视频播放器 + 简介 + 相关作品
关于页 → 账号介绍 + 联系方式
```

**技术实现**：
- WordPress + 视频主题
- 或静态站点（Hugo/Next.js）+ 视频托管

## ✅ 验收标准

### MVP 完成标志

- [ ] 输入一个话题，30 分钟内自动产出 1 分钟视频
- [ ] 视频质量：画面清晰、配音自然、字幕准确
- [ ] 网站可访问，至少展示 3 个示例作品
- [ ] 全流程无需手动干预（除首次配置）

### 质量指标

| 指标 | 目标值 |
|------|--------|
| 单视频生产时间 | <30 分钟 |
| 图片生成质量 | 用户满意度 >80% |
| 配音自然度 | 用户满意度 >75% |
| 视频剪辑流畅度 | 无明显卡顿/错位 |
